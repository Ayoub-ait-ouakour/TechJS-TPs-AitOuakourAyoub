# TPs — JavaScript et Mobile

Bref descriptif
- Ce dossier contient l’ensemble des travaux pratiques (TPs) pour le cours "JavaScript et Mobile".
- Chaque TP est livré dans son propre sous-dossier et rassemble le code source, les ressources et un README local expliquant l’exercice.

Organisation recommandée
- TP01-introduction/
- TP02-DOM-et-événements/
- TP03-API-et-asynchrone/
- TP04-PWA-et-mobile/
- TPNN-.../
- Chaque dossier : index.html (si applicable), scripts/, styles/, assets/, README.md

Conventions
- Nom de dossier : TP{numéro}-{sujet court}
- Inclure un README.md par TP avec :
    - Objectif
    - Étapes pour exécuter
    - Contraintes et points d’évaluation
    - Solutions ou corrigés (si fournis séparément, indiquer chemin)

Exécution et outils
- Interfaces web : ouvrir index.html dans le navigateur ou utiliser un serveur local (ex. Live Server, python -m http.server).
- Node.js : npm install && npm start (si le TP contient une partie back-end ou outils de build).
- Mobile / PWA : tester via navigateur mobile, émulateur ou appareil réel (activer le mode développeur si nécessaire).

Bonnes pratiques
- Commits atomiques et messages clairs (TP01 : ajout base, TP01 : correction exercice 2).
- Laisser le README du TP à jour avec les commandes nécessaires.
- Ne pas inclure de données sensibles.

Contact / aide
- Pour questions sur un TP, créer une issue en précisant le numéro du TP et une description concise du problème.
